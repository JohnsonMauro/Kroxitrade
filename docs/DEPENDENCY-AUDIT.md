# Dependency Audit

Audit date: 2026-07-16

Scope: direct dependencies declared in `package.json`. Transitive dependencies are not audited individually; they are controlled by their direct parent and `pnpm-lock.yaml`.

## Decision rule

Keep a dependency only when its maintenance and implementation cost is lower than a native alternative. Runtime dependencies deserve more scrutiny than development-only tooling. Before removing a dependency, preserve public formats and verify Chrome and Firefox builds.

## Recommended order

1. Remove `svgo` and verify `pnpm run typecheck` plus Chrome and Firefox builds.
2. Replace `js-base64` only with export/import compatibility fixtures in place.
3. Decide whether automatic import sorting justifies the Prettier plugin.
4. Keep Svelte/WXT and their supporting development dependencies unless a separately approved vanilla-extension migration is funded and planned.

## Audit evidence

- `pnpm why` confirms every entry above is a direct dependency of PoeTradePlus; Svelte and WXT are also required by `@wxt-dev/module-svelte`.
- Code search found `js-base64` only in bookmark export/import, 41 `lucide-static` imports, and no `svgo` source/config/script reference.
- `wxt.config.ts` registers `@wxt-dev/module-svelte`; `.prettierrc.mjs` registers the import-sorting plugin.
